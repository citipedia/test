題目一：祭祀圈中心選址分析

Q1: 若所有信眾皆從所屬祭祀圈的中心廟宇出發，哪一間媽祖廟作為折返點可最小化總直線距離？

假設每一間廟的信眾人數相同，讓所有人路走路總距離最短。我們要對所有信仰媽祖的廟宇計算距離矩陣，計算一間廟到所有其他廟的距離總和，找出最小總距離的那一間廟。

dist_matrix <- st_distance(mazu) #兩兩直線距離

optimal_mazu <- mazu[min_index, ]

Median Center

當目標是最小化總路徑長度、總成本移動距離。所有點位中，能使總直線距離總和最小化的點位，即為Median Center。

Mean Center

所有點位的 X 座標與 Y 座標的算術平均值所構成的點。當所有點位具有相同權重，且研究目的是最小化總體直線距離平方和時。

題目二：南部地區信仰的空間群聚特性

Q3: 比較南部地區中，主祭神祇為「媽祖」與「觀音菩薩」之寺廟的空間群聚特性。

將研究區域劃分成20km*20km網格，在每個格網中計算點的出現次數，統計所有格網的出現分布，若出現次數的變異程度遠高於隨機分布，即代表存在「空間群聚」現象

south_counties <- c("嘉義縣", "嘉義市", "台南市", "高雄市", "屏東縣")
mazu_south <- mazu %>% 
  filter(COUNTYNAME %in% south_counties)
gyps_south <- gyps %>% 
  filter(COUNTYNAME %in% south_counties)

#建立分析格網，計算每個格子內的點數
mazu_grid <- st_make_grid(mazu_south, cellsize = 20000, what = "polygons") %>% st_sf()
mazu_count <- st_contains(mazu_grid, mazu_south) %>% lengths()

gyps_grid <- st_make_grid(gyps_south, cellsize = 20000, what = "polygons") %>% st_sf()
gyps_count <- st_contains(gyps_grid, gyps_south) %>% lengths()

# VMR 與統計檢定
mazu_k <- length(mazu_count)
gyps_k <- length(gyps_count)

mazu_vmr <- var(mazu_count)/mean(mazu_count)
gyps_vmr <- var(gyps_count)/mean(gyps_count)

mazu_vmr_se <- sqrt(2 / (mazu_k - 1))
gyps_vmr_se <- sqrt(2 / (gyps_k - 1))

mazu_vmr_t <- (mazu_vmr - 1) / mazu_vmr_se
gyps_vmr_t <- (gyps_vmr - 1) / gyps_vmr_se

mazu_vmr_p <- 2 * pt(mazu_vmr_t, df = mazu_k - 1, lower.tail = mazu_vmr_t < 0)
gyps_vmr_p <- 2 * pt(gyps_vmr_t, df = gyps_k - 1, lower.tail = gyps_vmr_t < 0)

VMR的概念

VMR = 樣方內點數的變異數 / 樣方內點數的平均值

VMR 是衡量點位分布的集中或分散程度的統計指標，用來判斷點模式是否符合 Complete Spatial Randomness（CSR，完全空間隨機分布）。

VMR < 1 ：均勻（Regular）  VMR ≈ 1 ：隨機分布  VMR > 1 ：群聚（Clustered）

設定顯著水準 𝛼=0.05
H0：點分布為隨機分布(𝑠2=𝜆,𝑉𝑀𝑅=1)
H1：點分布非隨機分布(𝑠2≠𝜆,𝑉𝑀𝑅≠1)

因此，兩種寺廟的空間分布皆呈現顯著群聚現象。而媽祖廟的群聚程度遠高於觀音廟，顯示其在南部具有更明顯的地方集中特徵。媽祖信仰在空間上呈現較強的「聚落型」與「歷史集中」分布模式，可能與媽祖信仰的區域傳播、祭祀活動與文化地景緊密相關。

Q4: 討論樣方網格尺度對群聚分析的影響

比較 20km × 20km 與 50km × 50km 樣方下，媽祖廟空間分布的群聚統計特性，並討論格網尺度對檢定結果的影響。

mazu_grid_50 <- st_make_grid(mazu_south, cellsize = 50000) %>% st_sf()
count_50 <- st_contains(mazu_grid_50, mazu_south) %>% lengths()

k_50 <- length(count_50)
vmr_50 <- var(count_50) / mean(count_50)
vmr_se_50 <- sqrt(2 / (k_50 - 1))
vmr_t_50 <- (vmr_50 - 1) / vmr_se_50
p_50 <- 2 * pt(vmr_t_50, df = k_50 - 1, lower.tail = vmr_t_50 < 0)

雖然兩者的 p-value 都遠小於 0.05，顯示皆為顯著群聚分布，但我們觀察到：
  20km 樣方下的 VMR = 13.954，表現出強烈的局部聚集現象。  50km 樣方下的 VMR = 51.984，顯示高度集中於少數樣方。

這提醒我們：較小的網格尺度能揭露區域內更細緻的分布差異，適合用於觀察「聚落尺度」的信仰集中。較大的樣方則容易顯示整體性的大型集聚，例如某些都市或港口的宗教中心，但如果網格太大，部分細緻的聚集特性可能被稀釋或模糊。

在媽祖信仰的空間分布分析中，無論是 20km 或 50km 格網皆顯示顯著群聚，但格網大小會改變 VMR 統計數值與視覺判讀的尺度。空間尺度應依照研究目的彈性調整，或搭配多尺度分析以輔助解釋。
